This reference provides a comprehensive overview of Orchitect's configuration options and settings.

## Configuration File

Orchitect uses a configuration file to define project settings, provider connections, and platform behavior.

## Environment Variables

| Variable              | Description                | Default  |
| --------------------- | -------------------------- | -------- |
| `ORCHITECT_PORT`      | Port the server listens on | `8080`   |
| `ORCHITECT_LOG_LEVEL` | Logging verbosity          | `info`   |
| `ORCHITECT_DATA_DIR`  | Data storage directory     | `./data` |
